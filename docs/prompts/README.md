# Prompt Packs

This folder contains copy-ready prompt packs used to build HireProof with v0 and related agent tooling.

## Contents

- `v0-build-prompt-spec.md`: phased v0 prompt pack for scaffolding, demo fixtures, schemas, runtime MCP, SerpApi, AI SDK, fallback mode, polish, and deployment.

## Usage

Run the prompts in order and verify the app after each phase. Keep prompts focused on the core HireProof flow: paste a suspicious job post, run the investigation, and reveal a proof-backed verdict.
